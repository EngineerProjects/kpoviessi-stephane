NEXUS AI
AI Operating System / Autonomous Agent Platform

GitHub:
https://github.com/EngineerProjects/Nexus_ai

Description

Nexus AI est une plateforme expérimentale de nouvelle génération
conçue comme un véritable “AI Operating System”.

Le projet vise à permettre à des agents IA autonomes de collaborer,
exécuter des outils, gérer des workflows complexes,
maintenir une mémoire persistante et interagir avec
l’utilisateur dans un environnement desktop moderne.

Fonctionnalités principales

- Orchestration multi-agents
- Mémoire persistante et RAG
- Gestion des permissions et sandboxing
- Streaming temps réel via SSE
- Architecture modulaire basée sur des skills/plugins
- Communication gRPC et API REST
- Sessions persistantes
- Intégration MCP
- Interface desktop moderne

Architecture technique

- Backend Go haute performance
- API REST + gRPC
- Frontend Electron + React
- Système d’outils et plugins
- Vector databases et embeddings
- Architecture orientée agents

Technologies utilisées

Go, gRPC, PostgreSQL, SQLite, GORM,
React, Electron, TypeScript,
RAG, Vector Databases, SSE, MCP

Compétences démontrées

- AI Systems Engineering
- Software Architecture
- Multi-Agent Systems
- Backend Engineering
- Desktop Engineering
- AI Infrastructure
- Distributed Systems

TECH WATCH AGENT
Multi-Agent AI Research & Monitoring Platform

GitHub:
https://github.com/EngineerProjects/tech-watch-agent

Description

Tech Watch Agent est une plateforme autonome de veille
technologique et de recherche IA multi-agents permettant
d’automatiser des workflows complexes de monitoring,
d’analyse et de synthèse d’informations.

Le système peut rechercher automatiquement des sujets,
agréger plusieurs sources, analyser des contenus techniques
et générer des rapports structurés exploitables.

Fonctionnalités principales

- Recherche automatisée multi-sources
- Analyse GitHub, Reddit, arXiv et YouTube
- Génération automatique de rapports
- Synthèses IA structurées
- Veille continue programmable
- Pipelines RAG
- Notifications et envoi par email
- Sessions persistantes

Architecture technique

- Backend FastAPI
- Orchestration LangGraph
- PostgreSQL + pgvector
- Recherche via SearXNG
- Frontend React
- Dockerisation complète

Technologies utilisées

Python, FastAPI, LangGraph,
PostgreSQL, pgvector, Redis,
Docker, React, TypeScript, SearXNG

Compétences démontrées

- AI Engineering
- Retrieval-Augmented Generation (RAG)
- Multi-Agent Orchestration
- Backend APIs
- Data Pipelines
- Information Retrieval
- Fullstack Engineering
